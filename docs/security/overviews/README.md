# Security Overview Documents

This directory stores cross-cutting documents that are not tied to one imported
batch.

## Files

- `SECURITY_FINDINGS_RELATIONSHIP_2026-05-06.md`: initial relationship map.
- `SECURITY_FIX_QUEUE_2026-05-06.md`: original remediation queue.
- `SECURITY_FINDINGS_CURRENT_EXPOSURE_2026-05-06.md`: accepted exposure map.
- `SECURITY_FINDINGS_CLOSURE_REVIEW_2026-05-07.md`: closure status for the
  initial imported set.

Use `../README.md` for the current directory map and `../findings/README.md`
for canonical per-finding status.
